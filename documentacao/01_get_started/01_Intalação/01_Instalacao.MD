# Installation

## New Project

### Pre-Requisitos

* Node.js v18+
* IDE
* Terminal

Abra o terminal e use o seguinte comando:
```bash
npx nuxi@latest init <project-name> # Usei npx nuxi@latest init hello_world
```

Após a instalação finalizar, abra o seu projeto no terminal e rode o seguinte comando:

```bash
npm run dev -- -o
```
